// 001 — About the Course

Why start with a landing page?

A landing page is the perfect first project. It's small enough to finish in a weekend, yet rich enough to teach you the fundamentals that power every website on the internet.

You'll wrestle with real decisions — spacing, color, layout — and emerge with both a finished project and an intuition that no tutorial can hand you.

No frameworks. No shortcuts. Just HTML, CSS, and your creativity.

What you'll walk away with

01 —

A live, shareable project you built entirely from scratch.

02 —

Fluent HTML structure — sections, headers, footers, semantic elements.

03 —

CSS layout mastery — Flexbox, Grid, padding, and alignment patterns.

04 —

Designer's eye — color palettes, spacing systems, and visual hierarchy.

05 —

Confidence to start any project from a blank file.
